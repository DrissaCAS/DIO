Comunicação Client X Server

Início

Cliente - Consome a informação.

Servidor - 'Serve'/entrega as informações.

Cache - Arquivos gravados na máquina do cliente pelo navegador para evitar que seja necessário o envio de requisições ao servidor sem que tenha existido atualizações no site acessado.

F5 ou Refresh - Apaga os arquivos de cache salvos na máquina e busca as atualizações no servidor, ou melhor, reenvia a requisição de acesso ao servidor.

Design Responsivo - Diferentes resoluções para diferentes dispositivos.

Boas Práticas e Testes - Compatibilidade com vários navegadores.

História do HTML

Início

A linguagem HTML foi desenvolvida por Tim Berners-Lee e, na época, em torno de 1989, não se tratava de uma especificação, mas sim de uma forma de comunicação entre pesquisadores. Uma das primeiras funçõe das páginas web era apresentar currículos online de integrantes de universidades e centros de pesquisa americanos.

Tags

Início
  • Tag de Cabeçalhos HTML - Os cabeçalhos HTML são definidos com as tags < h1 > to < h6 >. < h1 > define o cabeçalho mais importante. < h6 > define o título menos importante
