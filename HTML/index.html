Comunicação Client X Server

Início

Cliente - Consome a informação.

Servidor - 'Serve'/entrega as informações.

Cache - Arquivos gravados na máquina do cliente pelo navegador para evitar que seja necessário o envio de requisições ao servidor sem que tenha existido atualizações no site acessado.

F5 ou Refresh - Apaga os arquivos de cache salvos na máquina e busca as atualizações no servidor, ou melhor, reenvia a requisição de acesso ao servidor.

Design Responsivo - Diferentes resoluções para diferentes dispositivos.

Boas Práticas e Testes - Compatibilidade com vários navegadores.

História do HTML

Início

A linguagem HTML é uma linguagem de marcação, desenvolvida por Tim Berners-Lee e, na época, em torno de 1989, não se tratava de uma especificação, mas sim de uma forma de comunicação entre pesquisadores. Uma das primeiras funçõe das páginas web era apresentar currículos online de integrantes de universidades e centros de pesquisa americanos.

Tags HTML

Início

      Tag de texto < h1 > to < h6 >

São tags de titulos, são definidos da tag < h1 > até < h6 >. < h1 > define o cabeçalho mais importante, com grafia mais robusta e visível. < h6 > define o título menos importante, com fonte menor.

      Tag de parágrafo < p >

Um parágrafo sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um parágrafo. A tag HTML < p > define um parágrafo.

      Tag de links < a >

Links são encontrados em quase todas as páginas da web. Os links permitem que os usuários cliquem de uma página para outra. Links HTML são hiperlinks. Você pode clicar em um link e pular para outro documento. Quando você move o mouse sobre um link, a seta do mouse se transforma em uma mãozinha.

Observação: um link não precisa ser texto. Um link pode ser uma imagem ou qualquer outro elemento HTML!

A tag HTML < a > define um hiperlink. O atributo mais importante do < a > elemento é o hrefatributo, que indica o destino do link. O texto do link é a parte que ficará visível para o leitor. Clicar no texto do link enviará o leitor ao endereço de URL especificado.

      Tag de listas < ul >, < li > e < ol >

As listas HTML permitem que os desenvolvedores da Web agrupem um conjunto de itens relacionados em listas.

      Lista HTML não ordenada

Uma lista não ordenada começa com a < ul > tag. Cada item da lista começa com a < li > tag.

      Lista HTML ordenada

Uma lista ordenada começa com a < ol > tag. Cada item da lista começa com a < li > tag.
